La commande SELECT en SQL

Introduction

La commande SELECT est utilisée pour extraire des données d'une base de données. C'est l'une des commandes les plus utilisées en SQL.

Syntaxe de base

SELECT colonnes FROM table;

Exemple :

SELECT nom, email FROM utilisateurs;

Utilisation de conditions

Vous pouvez filtrer les résultats avec WHERE :

SELECT * FROM utilisateurs WHERE age > 25;
