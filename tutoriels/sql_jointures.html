Les jointures en SQL

Les jointures permettent de combiner les données de plusieurs tables en fonction d’une condition commune. Elles sont essentielles pour travailler avec des bases de données relationnelles.

Les différents types de jointures

  • INNER JOIN : sélectionne les lignes avec des correspondances dans les deux tables.
  • LEFT JOIN : sélectionne toutes les lignes de la table de gauche, même sans correspondance dans la table de droite.
  • RIGHT JOIN : inverse du LEFT JOIN.
  • FULL JOIN : sélectionne toutes les lignes, qu’il y ait ou non correspondance.

Exemple de requête avec INNER JOIN


SELECT utilisateurs.nom, commandes.date
FROM utilisateurs
INNER JOIN commandes ON utilisateurs.id = commandes.utilisateur_id;
