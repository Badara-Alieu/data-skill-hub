Introduction à SQL

Pourquoi apprendre SQL ?

SQL (Structured Query Language) est le langage utilisé pour interagir avec les bases de données relationnelles. Il est indispensable pour analyser, extraire et manipuler les données dans des systèmes comme MySQL, PostgreSQL, Oracle ou SQL Server.

Les bases de SQL

Une base de données relationnelle est composée de tables. Une table ressemble à un tableau Excel : elle contient des lignes (enregistrements) et des colonnes (champs).

Exemple de table utilisateurs :


| id | prenom |      email      |
|----|--------|-----------------|
| 1  | Alice  | alice@mail.com  |
| 2  | Bob    | bob@mail.com    |

La requête suivante permet de sélectionner toutes les données :

SELECT * FROM utilisateurs;

Explication :

  • SELECT : pour lire des données
  • * : signifie "toutes les colonnes"
  • FROM utilisateurs : cible la table utilisateurs

Commandes SQL de base

  • SELECT : pour lire des données
  • INSERT : pour ajouter des données
  • UPDATE : pour modifier des données
  • DELETE : pour supprimer des données

Bonnes pratiques en SQL

  • Écrire les mots-clés SQL en majuscules (par convention)
  • Ne pas oublier le point-virgule en fin de requête
  • Faire attention aux noms de table et de colonne (ils peuvent être sensibles à la casse)
